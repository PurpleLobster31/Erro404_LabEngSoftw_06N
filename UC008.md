# Especificação de Caso de Uso: UC008

## Informações Gerais

| Campo | Conteúdo |
| :--- | :--- |
| **Identificador** | UC008 |
| **Nome** | Pesquisar Hospitais |
| **Atores** | Paciente |
| **Sumário** | O paciente realiza uma busca textual para encontrar um hospital específico. O resultado é uma lista filtrada que exibe as unidades correspondentes e seus respectivos tempos de atendimento. |
| **Pré-condição** | O sistema deve ter acesso à base de dados de unidades de saúde. |
| **Pós-condição** | O paciente visualiza os dados e tempos de espera das unidades pesquisadas. |
| **Pontos de Inclusão** | UC001 - Verificar Tempo em Pronto Atendimento |
| **Pontos de Extensão** | |

---

## Fluxo Principal

| Ações do Ator | Ações do Sistema |
| :--- | :--- |
| 1. O paciente acessa a funcionalidade de pesquisa de hospitais. | |
| | 2. O aplicativo exibe um campo de busca textual. |
| 3. O paciente insere o nome do hospital específico desejado e aciona o botão "Buscar". | |
| | 4. O sistema consulta a base de dados utilizando o termo fornecido. |
| | 5. O sistema atualiza a interface, exibindo uma lista filtrada apenas com as unidades que correspondem à busca, apresentando os tempos de atendimento de cada uma no mesmo formato da listagem geral. |

---

## Fluxo Alternativo: Busca Sem Resultados

| Ações do Ator | Ações do Sistema |
| :--- | :--- |
| 1. O paciente aciona o botão "Buscar" com um termo específico. | |
| | 2. O sistema não identifica nenhuma unidade correspondente ao termo na base de dados. |
| | 3. O aplicativo exibe uma mensagem informando que nenhum hospital foi encontrado. |
| | 4. O campo de busca é mantido disponível para a inserção de um novo termo. |

---

## Fluxo de Exceção 1: Indisponibilidade do Serviço de Busca

| Ações do Ator | Ações do Sistema |
| :--- | :--- |
| | 1. O sistema perde comunicação com a base de dados durante o processamento da pesquisa. |
| | 2. O aplicativo exibe uma mensagem de erro informando a indisponibilidade temporária da pesquisa. |
| | 3. A operação é interrompida. |
